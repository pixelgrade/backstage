Allow your visitors to access the Customizer and play with it.

== Description ==

Allow your visitors to access the Customizer without logging in. This way you can showcase the customization experience as close to reality as possible.

Everything is setup in such a way that people who are up to no good can't mess with your demo site data.

**Made with love by [Pixelgrade](https://pixelgrade.com)**

== Installation ==

Installing "Customizer Guest Demo Access" can be done either by searching for "Customizer Guest Demo Access" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org.
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Head over to Appearance -> Customizer Guest Access and set it up.

== Frequently Asked Questions ==

= Is there a way to reset the Customizer options to their default value? =
Reset buttons are available for all the options or for individual sections or panels.
This is mostly a development tool, thus it is disabled by default.
To enable them simply go to Dashboard -> Appearance -> Style Manager and check "Enable Reset Buttons"
